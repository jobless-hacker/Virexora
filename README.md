# Virexora - Client Acquisition Portfolio

SEO-focused, conversion-first portfolio website for Virexora, built as a static site for GitHub Pages.

## Included

- High-conversion service landing page
- Technical SEO essentials (metadata, schema, sitemap, robots)
- Mobile-first responsive design
- GitHub Pages compatible static architecture
- Multi-industry demo library for client pitching (`/demos/`)

## Deploy URL

After GitHub Pages is enabled:

https://jobless-hacker.github.io/Virexora/

## Customize before campaigns

- Update contact details in `index.html`
- Replace social profile URLs in JSON-LD
- Replace package pricing if needed
